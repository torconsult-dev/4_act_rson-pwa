Электронный акт обследования РСОН v0.9

Ссылка для GitHub Pages:
https://torconsult-dev.github.io/4_act_rson-pwa/

Как пользоваться:
1. На iPhone открывать через Safari, на Android через Chrome.
2. Для полного функционала разрешить доступ к геолокации, камере, микрофону, фото/файлам.
3. Если доступ не разрешён, данные можно вводить вручную.
4. Полный акт с фото, эскизом, подписями и QR формируется только через PDF: Печать / Сохранить как PDF.

Что изменено в v0.9:
- объединены варианты "Столбовая опора 300" и "Столбовая опора 600" в один тип акта "Столбовая опора";
- внутри акта добавлен выбор типа опоры: 300 / 600 / 1200;
- в кабельных линиях для кабель-канала добавлен выбор размера: 25х10, 25х25, 25х40, 40х40, другой вариант;
- материал стен теперь можно выбирать несколькими вариантами;
- добавлен материал стен "Сэндвич-панель";
- сохранены функции GPS, определение населённого пункта по GPS, фото, голосовые пояснения к фото, эскиз, ластик, отмена действия, подписи, QR и PDF.

Разработка и методическое сопровождение: проектно-сметный отдел ООО «ТОР-Консалт».
