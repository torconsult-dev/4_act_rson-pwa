Электронный акт обследования РСОН v0.11

Ссылка после публикации GitHub Pages:
https://torconsult-dev.github.io/4_act_rson-pwa/

Что изменено в v0.11:
- первая страница при запуске — выбор типа акта: ПУ / Мониторинг / Столбовая опора;
- выборы в форме переведены в компактные выпадающие списки;
- выбранные пункты показываются отдельной строкой под полем;
- мультивыбор материалов стен сделан компактным выпадающим блоком;
- сохранены GPS, населённый пункт по GPS, фото, пояснения к фото с голосовой надиктовкой, эскиз, ластик, отмена действия, подписи, QR, индивидуальный номер и PDF.

Для обновления GitHub Pages загрузить все файлы из архива в корень репозитория с заменой и нажать Commit changes.
После обновления на телефоне закрыть старую вкладку или очистить кэш, чтобы не подтягивалась старая версия.
