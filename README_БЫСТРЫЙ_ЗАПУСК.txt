Электронный акт РСОН v0.5

Что внутри:
- выбор типа акта на старте: ПУ, Мониторинг, Столбовая опора 300, Столбовая опора 600;
- структура полей повторяет приложенные бумажные акты;
- индивидуальный номер объекта формируется автоматически;
- QR-код выводится на первой странице печатного/PDF акта;
- GPS, фото, эскиз, заметки, подписи;
- полный акт формируется через PDF: Печать / Сохранить как PDF;
- отправка текстовой части на pavel.rus178@yandex.ru включена по умолчанию, галочку можно снять.

Публикация на GitHub Pages:
1. Распаковать архив.
2. Загрузить все файлы из папки в корень репозитория rson-pwa с заменой.
3. Нажать Commit changes.
4. Подождать 1-3 минуты.
5. Открыть https://torconsult-dev.github.io/rson-pwa/

Для полного функционала на телефоне открыть ссылку не внутри Telegram, а в Safari/Chrome и разрешить доступ к GPS, камере, микрофону и файлам. Если доступ не дать, данные можно вводить вручную.

Разработка и методическое сопровождение: проектно-сметный отдел ООО «ТОР-Консалт».
