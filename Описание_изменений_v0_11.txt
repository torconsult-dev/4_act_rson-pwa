Описание изменений релиза v0.11
Электронный акт обследования РСОН

Ссылка на приложение:
https://torconsult-dev.github.io/4_act_rson-pwa/

По сравнению с первым вариантом, где был один универсальный акт, приложение переведено на работу с несколькими типами актов:
- ПУ;
- Мониторинг;
- Столбовая опора.

Варианты «Столбовая опора 300» и «Столбовая опора 600» объединены в один тип акта «Столбовая опора». Тип опоры выбирается внутри акта: 300 / 600 / 1200.

Сохранены основные функции пилота: автоматические GPS-координаты, определение населённого пункта по GPS, фотофиксация, пояснения к фото с голосовой надиктовкой, голосовая надиктовка текстовых полей, эскиз, ластик, отмена последнего действия, подписи, индивидуальный номер объекта, QR-код и формирование полного PDF-акта.

В релизе v0.11 добавлено:
- при открытии приложения всегда первой загружается страница выбора типа акта;
- выборы в форме переведены в компактные выпадающие списки для уменьшения визуального объёма;
- выбранные пункты отображаются под строкой выбора;
- мультивыбор материалов стен сделан компактным выпадающим блоком;
- сохранён выбор размера кабель-канала: 25х10, 25х25, 25х40, 40х40, другой вариант;
- сохранён материал стен «сэндвич-панель».

Цель релиза — ускорить передачу данных обследования проектировщикам в цифровом виде. Дополнительные функции, включая единый реестр объектов, статусы, журнал замечаний, чат, склад, монтаж, ПНР и отчётность, целесообразно увязывать с перспективной CRM.
